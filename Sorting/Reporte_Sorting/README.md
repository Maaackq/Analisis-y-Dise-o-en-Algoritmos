# Reporte de sorting con medición experimental

Proyecto para analizar algoritmos de ordenamiento de VisuAlgo mediante implementación en Python, medición experimental y generación de gráficas.

## Estructura
- `src/`: código fuente de los algoritmos y benchmark
- `graficas/`: imágenes generadas automáticamente
- `reporte/`: reporte en formato Markdown

## Requisitos
Instala las dependencias con:

```bash
pip install -r requirements.txt
```

## Ejecución
Desde la terminal:

```bash
cd src
python benchmark.py
```

## Resultado
Las gráficas se guardan en la carpeta `graficas/`.
