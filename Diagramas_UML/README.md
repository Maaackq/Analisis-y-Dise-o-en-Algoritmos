# Diagramas UML de algoritmos de ordenamiento

En esta actividad creamos diagramas UML elaborados en PlantUML para los algoritmos de ordenamiento:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Random Quick Sort
- Counting Sort

## Tipo de UML utilizado

Se utilizaron **diagramas de actividad UML**, ya que permiten representar el flujo lógico de cada algoritmo, incluyendo condiciones, ciclos, decisiones y pasos de procesamiento.
